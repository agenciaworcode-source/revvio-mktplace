Banco de Dados

Migração do schema

3 tabelas novas em roxo. A tabela vehicles recebe apenas a FK seller_id.

Tabelas novas

sellers id uuid PK user_id uuid → auth.users name varchar(255) slug varchar(100) UNIQUE phone varchar(20) cpf_cnpj varchar(20) bio text avatar_url text banner_url text city / state varchar whatsapp varchar(20) instagram varchar(100) commission_rate decimal(5,2) → 5% status pending | active | suspended role seller | admin sales id uuid PK vehicle_id int → vehicles seller_id uuid → sellers buyer_name varchar(255) buyer_phone varchar(20) sale_price decimal(12,2) sale_date date payment_method pix | financiamento | à vista commissions id uuid PK sale_id uuid → sales seller_id uuid → sellers amount decimal(12,2) → valor R$ rate decimal(5,2) → % aplicado status pending | paid | overdue due_date date paid_at timestamptz

Tabela existente — alteração mínima

vehicles ← sem refactor id ... make, model ... price ... status ... owner_name ... ← mantido (proprietário do carro) owner_phone ... ← mantido images[] ... seller_id uuid → sellers ← NOVO Modelo de Negócio ← intermediação Owner = proprietário original do veículo Seller = vendedor na plataforma REVVIO Os dois conceitos coexistem ✓ RLS Policies ← automático vehicles: seller_id = current_seller() sales: seller_id = current_seller() commissions: seller_id = current_seller() + admin bypass policy

⚡ Migração não destrutiva

Adicionamos seller_id como coluna nullable, populamos com o vendedor padrão (admin atual) e só então ativamos o RLS. Zero downtime.

🔐

Fluxo de autenticação

1. Vendedor se cadastra em /cadastro-vendedor
2. Status fica como pending até admin aprovar
3. Admin aprova → status active
4. Vendedor faz login → redireciona para /painel
5. Admin faz login → redireciona para /dashboard
6. Middleware verifica role e seller_id em cada request

🔄

Owner vs Seller — Coexistência

Owner = proprietário original do veículo (dados do dono do carro)
Seller = vendedor intermediário na plataforma REVVIO

O modelo é intermediação: o seller cadastra o veículo de um owner e intermedia a venda. Ambos são entidades distintas no sistema.
